AI 生成題庫說明

本網站不串接 AI API，也不在前端放置任何 API Key。此頁只提供可複製的外部 AI 提示詞。

  1. 複製提示詞。
  2. 貼到 ChatGPT、Claude、Gemini 等 AI 工具。
  3. 上傳或貼上題庫來源資料。
  4. 請 AI 產生 JSON 題庫。
  5. 將 JSON 儲存成 .json 檔。
  6. 把 .json 放入 questions/ 資料夾，或使用網站的「匯入題庫」功能。
  7. 開始刷題。

AI 產生的題庫仍需人工檢查。官方題目與答案不可任意改寫。若原始資料有圖片，圖片檔要另外放入 assets/images/，JSON 裡的圖片路徑要與實際檔案位置一致。
